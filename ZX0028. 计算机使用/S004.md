# 28.4 Win10中新的控制台终端

更新日期: 2020/5/27

-----------------------------------------------------

## 1.	概述	

Windows操作系统中的命令行界面一直以丑著称。简陋的黑框白字、满地狗牙的文字、以及中文乱码等问题让人感觉难用。	

现在微软在Win10上推出了一款新的终端:	

- Windows Terminal

这可以说是一款时尚的现代化终端。它支持如下功能：	

- 多选项卡支持
- 分屏
- 高清字体
- 设置背景图片
- 背景毛玻璃效果
- 支持不同的命令行工具：CMD、WSL、Power Shell等
- 全Unicode支持
