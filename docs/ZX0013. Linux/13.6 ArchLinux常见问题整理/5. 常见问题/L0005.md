# 新用户无法执行sudo

更新日期：2022-12-17

---------------------------

Arch Linux中新建的用户没有sudo的权限。可以参考WiKi中的方法进行设置。

!!! warning "使用root用户执行"
    本章节的命令通常是在安装系统阶段使用root用户执行

## 1. 打开sudo编辑界面

```bash
EDITOR=vim visudo
```

!!! note "更换编辑工具"
    sudo默认使用vi进行编辑，这里我们指定使用更强大的vim来进行编辑。

## 2. 加入如下行来添加权限

    username ALL=(ALL) ALL
